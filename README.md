# 📊 Predictive Analytics Using Historical Data - Power BI Dashboard

## 📌 Project Overview

The **Predictive Analytics Using Historical Data** project is an interactive Power BI dashboard designed to analyze historical business data, identify trends, and forecast future performance. The dashboard enables data-driven decision-making through comprehensive visualizations, KPI tracking, and predictive analytics.

This project was developed as part of a **Data Analytics Internship** to demonstrate practical skills in **Business Intelligence, Data Visualization, Data Modeling, and Predictive Analytics** using Microsoft Power BI.

---

# 🎯 Project Objectives

* Analyze historical business performance.
* Forecast future sales trends using historical data.
* Monitor key business KPIs.
* Identify top-performing products, customers, and regions.
* Provide actionable business insights for decision-making.
